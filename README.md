# soc-triage-agent

An AI-powered Security Operations Center (SOC) Triage Agent built to assist security analysts in understanding and prioritizing security alerts.

## 🎯 Project Goal

The goal of this project is to develop an intelligent assistant that can:

* Analyze security alerts in plain English
* Explain what the alert means
* Assess the severity and potential impact
* Recommend investigation and response steps
* Reduce the time required for initial alert triage

## 🛠️ Tech Stack

* Git & GitHub
